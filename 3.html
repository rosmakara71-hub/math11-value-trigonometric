бһңбһ·бһүбҹ’бһүбһ¶бһҹбһ¶бһңбһ¶бһҹбҹӢбһҹбҹ’бһ‘бһ„бҹӢбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҹбһ·бһҹбҹ’бһҹ ( )
бһҡбҹҖбһ”бһҡбҹҖбһ„ бһ“бһ·бһ„бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҠбҹ„бһҷбҹ– бһӣбҹ„бһҖбһӮбҹ’бһҡбһј бһҡбһҹбҹӢ бһҳбһҖбһҡбһ¶

F

рҹ‘Ү бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҹбҹҶбһ“бһҪбһҡбһ“бһёбһҳбһҪбһҷбҹ—!

бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғбһҖбһ“бҹ’бһҹбҹ„бһҳбҹ– $\sin(4\pi - x)$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҳбһ»бҹҶбһҒбһҪбһ” $4\pi = 2(2\pi)$ бһўбһ¶бһ…бһӣбһ»бһ”бһ…бҹ„бһӣбһ”бһ¶бһ“ бһ“бһ¶бҹҶбһұбҹ’бһҷ $\sin(4\pi - x) = \sin(-x)$бҹ” бһҸбһ¶бһҳбһҳбһ»бҹҶбһ•бҹ’бһ‘бһ»бһҷ $\sin(-x) = -\sin x$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһӣбҹҒбһҒбһ“бҹғбҹ– $\cos(-30^\circ)$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһўбһ“бһ»бһӮбһҳбһ“бҹҚбһҖбһјбһҹбҹҠбһёбһ“бһ»бһҹбһҮбһ¶бһўбһ“бһ»бһӮбһҳбһ“бҹҚбһӮбһј бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҳбһ»бҹҶбһ•бҹ’бһ‘бһ»бһҷбҹ– $\cos(-30^\circ) = \cos(30^\circ) = \frac{\sqrt{3}}{2}$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғбһҖбһ“бҹ’бһҹбҹ„бһҳбҹ– $\tan(-x) + \tan x$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҳбһ»бҹҶбһ•бҹ’бһ‘бһ»бһҷ $\tan(-x) = -\tan x$бҹ” бһ“бһ¶бҹҶбһұбҹ’бһҷбһ”бһ¶бһ“бҹ– $-\tan x + \tan x = 0$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһӣбҹҒбһҒбһ“бҹғбҹ– $\sin(-60^\circ)$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҳбһ»бҹҶбһ•бҹ’бһ‘бһ»бһҷбҹ– $\sin(-60^\circ) = -\sin(60^\circ) = -\frac{\sqrt{3}}{2}$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғбһҖбһ“бҹ’бһҹбҹ„бһҳбҹ– $\sin(90^\circ - A) - \cos A$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҳбһ»бҹҶбһ”бҹҶбһ–бҹҒбһүбҹ– $\sin(90^\circ - A) = \cos A$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮ $\cos A - \cos A = 0$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғбһ•бһӣбһ’бҹҖбһ”бҹ– $\frac{\tan(25^\circ)}{\cot(65^\circ)}$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҠбҹ„бһҷбһҹбһ¶бһҡ $25^\circ + 65^\circ = 90^\circ$ бһ“бһ¶бҹҶбһұбҹ’бһҷбһҳбһ»бҹҶбһ”бҹҶбһ–бҹҒбһү $\cot(65^\circ) = \tan(25^\circ)$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһ•бһӣбһ’бҹҖбһ”бһҹбҹ’бһҳбһҫ $1$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғбһҖбһ“бҹ’бһҹбҹ„бһҳбҹ– $\cos\left(\frac{\pi}{2} - \theta\right)$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҳбһ»бҹҶбһ”бҹҶбһ–бҹҒбһү бһҖбһјбһҹбҹҠбһёбһ“бһ»бһҹбһ”бҹ’бһҸбһјбһҡбһ‘бҹ…бһҮбһ¶бһҹбҹҠбһёбһ“бһ»бһҹ бһ“бһ¶бҹҶбһұбҹ’бһҷ $\cos\left(\frac{\pi}{2} - \theta\right) = \sin\theta$бҹ”
бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ“бҹ’бһҹбҹ„бһҳбҹ– $\sin^2(15^\circ) + \sin^2(75^\circ)$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҠбҹ„бһҷбһҹбһ¶бһҡ $15^\circ + 75^\circ = 90^\circ$ бһ“бһ¶бҹҶбһұбҹ’бһҷ $\sin(75^\circ) = \cos(15^\circ)$бҹ” бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһӮбҹ’бһҡбһ№бҹҮ $\sin^2(15^\circ) + \cos^2(15^\circ) = 1$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ–бһ·бһҸбһ“бҹғбҹ– $\sin(\pi - \alpha)$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҳбһ»бҹҶбһ”бһ“бҹ’бһҗбҹӮбһҳ бһҹбҹҠбһёбһ“бһ»бһҹбһ“бҹғбһҳбһ»бҹҶбһ”бһ“бҹ’бһҗбҹӮбһҳбһҳбһ¶бһ“бһҹбһүбҹ’бһүбһ¶бһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“бһҠбһјбһ…бһӮбҹ’бһ“бһ¶ бһ“бһ¶бҹҶбһұбҹ’бһҷ $\sin(\pi - \alpha) = \sin\alpha$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһӣбҹҒбһҒбһ“бҹғбҹ– $\cos(135^\circ)$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҹбһҡбһҹбҹҒбһҡ $135^\circ = 180^\circ - 45^\circ$бҹ” бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҳбһ»бҹҶбһ”бһ“бҹ’бһҗбҹӮбһҳбҹ– $\cos(180^\circ - 45^\circ) = -\cos(45^\circ) = -\frac{\sqrt{2}}{2}$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһӣбҹҒбһҒбһ“бҹғбҹ– $\tan(120^\circ)$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҹбһҡбһҹбҹҒбһҡ $120^\circ = 180^\circ - 60^\circ$бҹ” бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҳбһ»бҹҶбһ”бһ“бҹ’бһҗбҹӮбһҳбҹ– $\tan(180^\circ - 60^\circ) = -\tan(60^\circ) = -\sqrt{3}$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғбһҖбһ“бҹ’бһҹбҹ„бһҳбҹ– $\sin(\pi + \theta)$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҳбһ»бҹҶбһҳбһ¶бһ“бһ•бһӣбһҹбһ„бһҹбҹ’бһҳбһҫ $\pi$бҹ– бһҳбһ»бҹҶбһҹбҹ’бһҗбһ·бһҸбһ“бҹ…бһҖбһ¶бһҠбҹ’бһҡбһ„бҹӢбһ‘бһёбҹЈ бһ“бһ¶бҹҶбһұбҹ’бһҷбһҸбһҳбҹ’бһӣбҹғбһҹбҹҠбһёбһ“бһ»бһҹбһўбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“ $\sin(\pi + \theta) = -\sin\theta$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһӣбҹҒбһҒбһ“бҹғбҹ– $\cos(210^\circ)$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҹбһҡбһҹбҹҒбһҡ $210^\circ = 180^\circ + 30^\circ$бҹ” бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбҹ– $\cos(180^\circ + 30^\circ) = -\cos(30^\circ) = -\frac{\sqrt{3}}{2}$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһӣбҹҒбһҒбһ“бҹғбҹ– $\tan(225^\circ)$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҹбһҡбһҹбҹҒбһҡ $225^\circ = 180^\circ + 45^\circ$бҹ” бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбҹ– $\tan(180^\circ + 45^\circ) = \tan(45^\circ) = 1$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғбһҖбһ“бҹ’бһҹбҹ„бһҳбҹ– $\sin\left(\frac{\pi}{2} + \alpha\right)$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҳбһ»бҹҶбһ•бһӣбһҹбһ„бһҹбҹ’бһҳбһҫ $\frac{\pi}{2}$ бһҹбҹҠбһёбһ“бһ»бһҹбһ“бҹ…бһҖбһ¶бһҠбҹ’бһҡбһ„бҹӢбһ‘бһёбҹў бһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“ бһ“бһ¶бҹҶбһұбҹ’бһҷ $\sin\left(\frac{\pi}{2} + \alpha\right) = \cos\alpha$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғбһҖбһ“бҹ’бһҹбҹ„бһҳбҹ– $\cos\left(\frac{\pi}{2} + x\right)$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҳбһ»бҹҶбһ•бһӣбһҹбһ„бһҹбҹ’бһҳбһҫ $\frac{\pi}{2}$ бһҖбһјбһҹбҹҠбһёбһ“бһ»бһҹбһ“бҹ…бһҖбһ¶бһҠбҹ’бһҡбһ„бҹӢбһ‘бһёбҹў бһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһўбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“ бһ“бһ¶бҹҶбһұбҹ’бһҷ $\cos\left(\frac{\pi}{2} + x\right) = -\sin x$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһӣбҹҒбһҒбһ“бҹғбҹ– $\sin(150^\circ)$ бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҳбһ»бҹҶбһ•бһӣбһҹбһ„бһҹбҹ’бһҳбһҫ $90^\circ$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҹбһҡбһҹбҹҒбһҡ $150^\circ = 90^\circ + 60^\circ$бҹ” бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбҹ– $\sin(90^\circ + 60^\circ) = \cos(60^\circ) = \frac{\sqrt{3}}{2}$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһӣбҹҒбһҒбһ“бҹғбҹ– $\cos(390^\circ)$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҹбһҡбһҹбҹҒбһҡ $390^\circ = 360^\circ + 30^\circ$бҹ” бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҳбһ»бҹҶбһҒбһҪбһ”бҹ– $\cos(360^\circ + 30^\circ) = \cos(30^\circ) = \frac{\sqrt{3}}{2}$бҹ”
бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ“бҹ’бһҹбҹ„бһҳбҹ– $\sin(-x) \times \cot(-x)$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһҳбһ»бҹҶбһ•бҹ’бһ‘бһ»бһҷбҹ– $\sin(-x) = -\sin x$ бһ“бһ·бһ„ $\cot(-x) = -\cot x$бҹ” бһ“бһ¶бҹҶбһұбҹ’бһҷбһ”бһ¶бһ“ $(-\sin x) \times (-\cot x) = \sin x \times \frac{\cos x}{\sin x} = \cos x$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғбһ•бһӣбһ”бһјбһҖбҹ– $\cos(40^\circ) + \cos(140^\circ)$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҠбҹ„бһҷбһҹбһ¶бһҡ $40^\circ + 140^\circ = 180^\circ$ бһ“бһ¶бҹҶбһұбҹ’бһҷбһҳбһ»бҹҶбһ”бһ“бҹ’бһҗбҹӮбһҳ $\cos(140^\circ) = -\cos(40^\circ)$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮ $\cos(40^\circ) + (-\cos(40^\circ)) = 0$бҹ”
бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ“бҹ’бһҹбҹ„бһҳбҹ– $\tan\left(\frac{\pi}{2} - x\right) \times \tan x$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҳбһ»бҹҶбһ”бҹҶбһ–бҹҒбһүбҹ– $\tan\left(\frac{\pi}{2} - x\right) = \cot x$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮ $\cot x \times \tan x = 1$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһӣбҹҒбһҒбһ“бҹғбҹ– $\sin(240^\circ)$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҹбһҡбһҹбҹҒбһҡ $240^\circ = 180^\circ + 60^\circ$бҹ” бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбҹ– $\sin(180^\circ + 60^\circ) = -\sin(60^\circ) = -\frac{\sqrt{3}}{2}$бҹ”
бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ“бҹ’бһҹбҹ„бһҳбҹ– $\cos(2\pi - \alpha) - \cos\alpha$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҳбһ»бҹҶбһҒбһҪбһ” бһ“бһ·бһ„бһҳбһ»бҹҶбһ•бҹ’бһ‘бһ»бһҷбҹ– $\cos(2\pi - \alpha) = \cos(-\alpha) = \cos\alpha$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮ $\cos\alpha - \cos\alpha = 0$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ–бһ·бһҸбһ“бҹғбҹ– $\sin(100^\circ) - \sin(80^\circ)$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҠбҹ„бһҷбһҹбһ¶бһҡ $100^\circ + 80^\circ = 180^\circ$ (бһҳбһ»бҹҶбһ”бһ“бҹ’бһҗбҹӮбһҳ) бһ“бһ¶бҹҶбһұбҹ’бһҷ $\sin(100^\circ) = \sin(80^\circ)$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһҖбһ“бҹ’бһҹбҹ„бһҳбһҹбҹ’бһҳбһҫ $0$бҹ”
бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ“бҹ’бһҹбҹ„бһҳбҹ– $\sin\left(\frac{\pi}{2} + x\right) + \cos(\pi - x)$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбҹ– $\sin\left(\frac{\pi}{2} + x\right) = \cos x$ бһ“бһ·бһ„ $\cos(\pi - x) = -\cos x$бҹ” бһ“бһ¶бҹҶбһұбҹ’бһҷбһ”бһ¶бһ“бҹ– $\cos x + (-\cos x) = 0$бҹ”
