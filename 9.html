бһңбһ·бһүбҹ’бһүбһ¶бһҹбһ¶бһңбһ¶бһҹбҹӢбһҹбҹ’бһ‘бһ„бҹӢбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҹбһ·бһҹбҹ’бһҹ ( )

F

рҹ‘Ү бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҹбҹҶбһ“бһҪбһҡбһ“бһёбһҳбһҪбһҷбҹ—!

бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғбһҖбһ“бҹ’бһҹбҹ„бһҳбҹ– $\tan(-x) - \tan(2\pi - x)$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһҳбһ»бҹҶбһ•бҹ’бһ‘бһ»бһҷ $\tan(-x) = -\tan x$бҹ” бһҸбһ¶бһҳбһҳбһ»бҹҶбһҒбһҪбһ” $\tan(2\pi - x) = \tan(-x) = -\tan x$бҹ” бһ“бһ¶бҹҶбһұбҹ’бһҷбһ”бһ¶бһ“бҹ– $(-\tan x) - (-\tan x) = 0$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһӣбҹҒбһҒбһ“бҹғбҹ– $\csc(-30^\circ)$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҠбҹ„бһҷбһҹбһ¶бһҡ $\csc\theta = \frac{1}{\sin\theta}$ бһ бһҫбһҷбһҹбҹҠбһёбһ“бһ»бһҹбһҮбһ¶бһўбһ“бһ»бһӮбһҳбһ“бҹҚбһҹбҹҒбһҹ бһ“бһ¶бҹҶбһұбҹ’бһҷ $\csc(-30^\circ) = -\csc(30^\circ) = -\frac{1}{\sin(30^\circ)} = -\frac{1}{1/2} = -2$бҹ”
бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ“бҹ’бһҹбҹ„бһҳбҹ– $\sin(-x) \times \csc x$ бһ”бһҫ $\sin x \neq 0$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһҳбһ»бҹҶбһ•бҹ’бһ‘бһ»бһҷ $\sin(-x) = -\sin x$бҹ” бһ“бһ¶бҹҶбһұбҹ’бһҷбһ”бһ¶бһ“бҹ– $(-\sin x) \times \csc x = (-\sin x) \times \frac{1}{\sin x} = -1$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһӣбҹҒбһҒбһ“бҹғбҹ– $\cos(-60^\circ) + \cos(60^\circ)$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһҳбһ»бҹҶбһ•бҹ’бһ‘бһ»бһҷ $\cos(-60^\circ) = \cos(60^\circ) = \frac{1}{2}$бҹ” бһ“бһ¶бҹҶбһұбҹ’бһҷбһ”бһ¶бһ“бҹ– $\frac{1}{2} + \frac{1}{2} = 1$бҹ”
бһ”бһҫ $A$ бһ“бһ·бһ„ $B$ бһҮбһ¶бһҳбһ»бҹҶбһ”бҹҶбһ–бҹҒбһүбһӮбҹ’бһ“бһ¶ ($A+B=90^\circ$) бһ“бҹ„бҹҮ $\sin A - \cos B$ бһҹбҹ’бһҳбһҫбһ“бһ№бһ„бҹ– (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҠбҹ„бһҷбһҹбһ¶бһҡ $A+B=90^\circ \Rightarrow B = 90^\circ - A$бҹ” бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҳбһ»бҹҶбһ”бҹҶбһ–бҹҒбһү $\cos(90^\circ - A) = \sin A$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮ $\sin A - \sin A = 0$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһӣбҹҒбһҒбһ“бҹғбҹ– $\cot(35^\circ) \times \cot(55^\circ)$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҠбҹ„бһҷбһҹбһ¶бһҡ $35^\circ + 55^\circ = 90^\circ$ бһ“бһ¶бҹҶбһұбҹ’бһҷбһ”бһ¶бһ“бһҳбһ»бҹҶбһ”бҹҶбһ–бҹҒбһү $\cot(55^\circ) = \tan(35^\circ)$бҹ” бһҸбһ¶бһҳбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһӮбҹ’бһҡбһ№бҹҮ $\cot(35^\circ) \times \tan(35^\circ) = 1$бҹ”
бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ“бҹ’бһҹбҹ„бһҳбҹ– $\cot(90^\circ - x) \times \cot x$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҳбһ»бҹҶбһ”бҹҶбһ–бҹҒбһү $\cot(90^\circ - x) = \tan x$бҹ” бһ“бһ¶бҹҶбһұбҹ’бһҷбһ”бһ¶бһ“ $\tan x \times \cot x = 1$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһӣбҹҒбһҒбһ“бҹғбҹ– $\sin^2(1^\circ) + \sin^2(89^\circ)$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҠбҹ„бһҷбһҹбһ¶бһҡ $1^\circ + 89^\circ = 90^\circ$ бһ“бһ¶бҹҶбһұбҹ’бһҷ $\sin(89^\circ) = \cos(1^\circ)$бҹ” бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһӮбҹ’бһҡбһ№бҹҮбҹ– $\sin^2(1^\circ) + \cos^2(1^\circ) = 1$бҹ”
бһ”бһҫ $A, B, C$ бһҮбһ¶бһҳбһ»бҹҶбһҖбҹ’бһ“бһ»бһ„бһ“бҹғбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһҪбһҷ бһ“бҹ„бҹҮ $\sin(A+B)$ бһҹбҹ’бһҳбһҫбһ“бһ№бһ„бҹ– (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҖбҹ’бһ“бһ»бһ„бһҸбҹ’бһҡбһёбһҖбҹ„бһҺ $A+B+C = 180^\circ \Rightarrow A+B = 180^\circ - C$бҹ” бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҳбһ»бҹҶбһ”бһ“бҹ’бһҗбҹӮбһҳбҹ– $\sin(180^\circ - C) = \sin C$бҹ”
бһ”бһҫ $A, B, C$ бһҮбһ¶бһҳбһ»бҹҶбһҖбҹ’бһ“бһ»бһ„бһ“бҹғбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһҪбһҷ бһ“бҹ„бҹҮ $\cos(A+B) + \cos C$ бһҹбҹ’бһҳбһҫбһ“бһ№бһ„бҹ– (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҠбҹ„бһҷбһҹбһ¶бһҡ $A+B = 180^\circ - C$ бһ“бһ¶бҹҶбһұбҹ’бһҷбһ”бһ¶бһ“ $\cos(A+B) = \cos(180^\circ - C) = -\cos C$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮ $-\cos C + \cos C = 0$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһӣбҹҒбһҒбһ“бҹғбҹ– $\sin^2(45^\circ) + \cos^2(135^\circ)$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһҳбһ»бҹҶбһ”бһ“бҹ’бһҗбҹӮбһҳ $\cos(135^\circ) = -\cos(45^\circ) \Rightarrow \cos^2(135^\circ) = \cos^2(45^\circ)$бҹ” бһ“бһ¶бҹҶбһұбҹ’бһҷбһ”бһ¶бһ“бҹ– $\sin^2(45^\circ) + \cos^2(45^\circ) = 1$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғбһҖбһ“бҹ’бһҹбҹ„бһҳбҹ– $\csc(\pi + x)$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҳбһ»бҹҶбһҳбһ¶бһ“бһ•бһӣбһҹбһ„бһҹбҹ’бһҳбһҫ $\pi$бҹ– $\sin(\pi + x) = -\sin x$ бһ“бһ¶бҹҶбһұбҹ’бһҷбһ”бһ¶бһ“ $\csc(\pi + x) = \frac{1}{\sin(\pi + x)} = \frac{1}{-\sin x} = -\csc x$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһӣбҹҒбһҒбһ“бҹғбҹ– $\tan(210^\circ) \times \cot(210^\circ)$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһ‘бҹ„бҹҮбһ”бһёбһҮбһ¶бһҳбһ»бҹҶбһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“бһҖбҹҸбһҠбҹ„бһҷ (бһұбҹ’бһҷбһҸбҹӮбһўбһ“бһ»бһӮбһҳбһ“бҹҚбһҳбһ¶бһ“бһ“бҹҗбһҷ) бһ•бһӣбһӮбһ»бһҺбһҡбһңбһ¶бһ„бһҸбһ„бҹӢбһҹбһ„бҹӢ бһ“бһ·бһ„бһҖбһјбһҸбһ„бҹӢбһҹбһ„бҹӢбһ“бҹғбһҳбһ»бҹҶбһҠбһҠбҹӮбһӣбһҮбһ¶бһ“бһ·бһ…бҹ’бһ…бһҖбһ¶бһӣбһҹбҹ’бһҳбһҫ $1$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһӣбҹҒбһҒбһ“бҹғбҹ– $\sin(240^\circ)$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҹбһҡбһҹбҹҒбһҡ $240^\circ = 180^\circ + 60^\circ$бҹ” бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбҹ– $\sin(180^\circ + 60^\circ) = -\sin(60^\circ) = -\frac{\sqrt{3}}{2}$бҹ”
бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ“бҹ’бһҹбҹ„бһҳбҹ– $\cot\left(\frac{\pi}{2} + x\right) + \tan x$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҳбһ»бҹҶбһ•бһӣбһҹбһ„бһҹбҹ’бһҳбһҫ $\frac{\pi}{2}$бҹ– $\cot\left(\frac{\pi}{2} + x\right) = -\tan x$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮ $-\tan x + \tan x = 0$бҹ”
бһ”бһҫ $A, B, C$ бһҮбһ¶бһҳбһ»бҹҶбһҖбҹ’бһ“бһ»бһ„бһ“бҹғбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһҪбһҷ бһ“бҹ„бҹҮ $\tan\left(\frac{A+B}{2}\right)$ бһҹбҹ’бһҳбһҫбһ“бһ№бһ„бҹ– (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҠбҹ„бһҷбһҹбһ¶бһҡ $A+B+C = 180^\circ \Rightarrow \frac{A+B}{2} = 90^\circ - \frac{C}{2}$бҹ” бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҳбһ»бҹҶбһ”бҹҶбһ–бҹҒбһүбҹ– $\tan\left(90^\circ - \frac{C}{2}\right) = \cot\left(\frac{C}{2}\right)$бҹ”
бһ”бһҫ $A, B, C$ бһҮбһ¶бһҳбһ»бҹҶбһҖбҹ’бһ“бһ»бһ„бһ“бҹғбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһҪбһҷ бһ“бҹ„бҹҮ $\cos\left(\frac{A+B}{2}\right)$ бһҹбҹ’бһҳбһҫбһ“бһ№бһ„бҹ– (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһҠбҹӮбһҡ $\frac{A+B}{2} = 90^\circ - \frac{C}{2}$бҹ” бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҳбһ»бҹҶбһ”бҹҶбһ–бҹҒбһүбҹ– $\cos\left(90^\circ - \frac{C}{2}\right) = \sin\left(\frac{C}{2}\right)$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһӣбҹҒбһҒбһ“бҹғбҹ– $\sin(-120^\circ)$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһҳбһ»бҹҶбһ•бҹ’бһ‘бһ»бһҷ $\sin(-120^\circ) = -\sin(120^\circ)$бҹ” бһҸбһ¶бһҳбһҳбһ»бҹҶбһ”бһ“бҹ’бһҗбҹӮбһҳ $-\sin(120^\circ) = -\sin(180^\circ - 60^\circ) = -\sin(60^\circ) = -\frac{\sqrt{3}}{2}$бҹ”
бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ“бҹ’бһҹбҹ„бһҳбҹ– $\frac{\cos(\pi - x)}{\sin\left(\frac{\pi}{2} + x\right)}$ бһ”бһҫ $\cos x \neq 0$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһҳбһ»бҹҶбһ”бһ“бҹ’бһҗбҹӮбһҳ $\cos(\pi - x) = -\cos x$ бһ“бһ·бһ„бһҸбһ¶бһҳбһҳбһ»бҹҶбһ•бһӣбһҹбһ„бһҹбҹ’бһҳбһҫ $\frac{\pi}{2}$бҹ– $\sin\left(\frac{\pi}{2} + x\right) = \cos x$бҹ” бһ“бһ¶бҹҶбһұбҹ’бһҷбһ•бһӣбһ”бҹ’бһҡбһ—бһ¶бһӮбһҹбҹ’бһҳбһҫ $\frac{-\cos x}{\cos x} = -1$бҹ”
бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ“бҹ’бһҹбҹ„бһҳбҹ– $\sin\left(\frac{3\pi}{2} - x\right)$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҹбһҡбһҹбҹҒбһҡ $\frac{3\pi}{2} = \pi + \frac{\pi}{2}$бҹ” бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбҹ– $\sin\left(\pi + \left(\frac{\pi}{2} - x\right)\right) = -\sin\left(\frac{\pi}{2} - x\right) = -\cos x$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғбһ•бһӣбһ”бһјбһҖбҹ– $\tan(10^\circ) + \tan(50^\circ) + \tan(130^\circ) + \tan(170^\circ)$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһ•бҹ’бһӮбһјбһ•бҹ’бһӮбһ„бһҳбһ»бҹҶбһ”бһ“бҹ’бһҗбҹӮбһҳбҹ– $\tan(170^\circ) = -\tan(10^\circ)$ бһ“бһ·бһ„ $\tan(130^\circ) = -\tan(50^\circ)$бҹ” бһ“бһ¶бҹҶбһұбҹ’бһҷбһ”бһ¶бһ“бҹ– $(\tan 10^\circ - \tan 10^\circ) + (\tan 50^\circ - \tan 50^\circ) = 0$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ•бһӣбһ’бҹҖбһ”бҹ– $\frac{\sin(-x)}{\sin(\pi + x)}$ бһ”бһҫ $\sin x \neq 0$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһҳбһ»бҹҶбһ•бҹ’бһ‘бһ»бһҷ $\sin(-x) = -\sin x$бҹ” бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸ $\sin(\pi + x) = -\sin x$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһ•бһӣбһ’бҹҖбһ”бһҹбҹ’бһҳбһҫ $\frac{-\sin x}{-\sin x} = 1$бҹ”
бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ“бҹ’бһҹбҹ„бһҳбҹ– $\cos\left(\frac{7\pi}{2} - x\right)$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҹбһҡбһҹбҹҒбһҡ $\frac{7\pi}{2} = 4\pi - \frac{\pi}{2}$бҹ” бһҸбһ¶бһҳбһҳбһ»бҹҶбһҒбһҪбһ” бһўбһ¶бһ…бһӣбһ»бһ” $4\pi$ бһ”бһ¶бһ“ бһ“бһ¶бҹҶбһұбҹ’бһҷбһ”бһ¶бһ“ $\cos\left(-\frac{\pi}{2} - x\right) = \cos\left(\frac{\pi}{2} + x\right)$ (бһ–бҹ’бһҡбҹ„бҹҮбһҖбһјбһҹбҹҠбһёбһ“бһ»бһҹбһҮбһ¶бһўбһ“бһ»бһӮбһҳбһ“бҹҚбһӮбһј)бҹ” бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбҹ– $\cos\left(\frac{\pi}{2} + x\right) = -\sin x$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғбҹ– $\cos(120^\circ) \times \cos(240^\circ)$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһҳбһ»бҹҶбһ”бһ“бҹ’бһҗбҹӮбһҳ $\cos(120^\circ) = -\cos(60^\circ) = -\frac{1}{2}$бҹ” бһҸбһ¶бһҳбһҳбһ»бҹҶбһҳбһ¶бһ“бһ•бһӣбһҹбһ„бһҹбҹ’бһҳбһҫ $\pi$бҹ– $\cos(240^\circ) = -\cos(60^\circ) = -\frac{1}{2}$бҹ” бһ“бһ¶бҹҶбһұбҹ’бһҷбһ”бһ¶бһ“бҹ– $\left(-\frac{1}{2}\right) \times \left(-\frac{1}{2}\right) = \frac{1}{4}$бҹ”
бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ“бҹ’бһҹбҹ„бһҳбҹ– $\frac{1}{1 + \tan^2 x} + \frac{1}{1 + \cot^2 x}$ (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһӮбҹ’бһҡбһ№бҹҮбҹ– $1 + \tan^2 x = \sec^2 x \Rightarrow \frac{1}{\sec^2 x} = \cos^2 x$ бһ“бһ·бһ„ $1 + \cot^2 x = \csc^2 x \Rightarrow \frac{1}{\csc^2 x} = \sin^2 x$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһҖбһ“бҹ’бһҹбҹ„бһҳбһ‘бҹ…бһҮбһ¶бҹ– $\cos^2 x + \sin^2 x = 1$бҹ”
